# Acceptance Rejection Comparison Output Fidelity Notes

This file records target output requirements for later production. It does not claim a rendered asset passes.

## Canvas

CH06 comparison specs declare `1100 x 620`. This grammar treats that as the candidate CH06 canvas and flags protocol `1280` full-column reconciliation as a pre-Layer-3 Design Authority decision if strict standardization is required.

## Print Readiness

- Shared setup and outcome labels must remain legible at book size.
- Compared panels must remain readable without color.
- Shared scale indicators must not disappear in print.
- The no-trade strip, when present, must not become a tiny footnote.

## Output Boundary

No PNG is authoritative. Future SVG/PDF assets are the source of truth. PNG review is a later human-review aid and was not run for this package.

## Metadata

Future assets must carry spec version, grammar version, renderer identity, source commit, and human approval records only after real sign-off. Candidate files must never imply approval.
