# Acceptance Rejection Comparison Layout Rules

This candidate grammar defines comparison layouts where the same start resolves into distinct reads.

## Canonical Composition

1. Title band with comparison name and teaching claim.
2. Shared setup band across the top of the body.
3. Left outcome panel.
4. Right outcome panel.
5. Optional decision strip for no-trade or classification state.
6. Evidence rail on the right.
7. Forbidden rail at bottom or below the evidence rail.
8. Caption and footer.

The shared setup is visually upstream of both outcomes. Outcome panels must not appear to start from different conditions.

## Named Regions

- `title-band`
- `shared-setup`
- `left-outcome`
- `right-outcome`
- `decision-strip`
- `shared-scale-badge`
- `evidence-rail`
- `forbidden-rail`
- `caption-footer`

Producers fill these regions without inventing local variants.

## Shared Setup Rules

- Draw one watched level or reclaim level.
- Draw one entrant or trapped population zone.
- Draw the common impulse or break once.
- Draw the stop pool or buy-stop pool location before activation.
- Add one badge stating "same setup, read decided after."

The outcome panels inherit this setup. They do not redraw different starting conditions.

## Outcome Panel Rules

Each outcome panel must carry:

- a direct branch title;
- one deciding evidence mark;
- one consequence mark;
- one invalidated misread guard if needed.

For defended breakout:

- hold above the level;
- bid refresh and sell absorption;
- value migration above the level.

For trapped breakout:

- failure back through the level;
- reentry print with speed and sell volume;
- forced-exit path to finite target.

For covering-only:

- first pullback loses the reclaim level;
- no bid rebuild;
- no value migration;
- rally exhausts after covering finishes.

For demand-confirmed:

- first pullback is shallow and holds;
- bids refresh;
- two-sided trade builds;
- renewed initiative buying and value migration appear.

For no-trade:

- use a neutral decision strip with unresolved/mixed evidence;
- do not draw no-trade as a weak third chart unless approved.

## Shared Scale

Compared panels use identical price scale unless the spec explicitly permits otherwise. For `CH06_DEFENDED_BREAKOUT_VS_TRAPPED_BREAKOUT_001`, shared scale is required evidence and must be visible in the body.

## Mark Vocabulary

- Shared level: dashed price line with monospace badge.
- Entry or short-entry zone: bracketed band.
- Stop pool or buy-stop pool: stacked markers on the activation side.
- Impulse: thick path plus compact volume/delta glyph.
- First-pullback test: bracket or node between shared setup and outcomes.
- Value migration: stacked horizontal value bars or developing distribution block.
- Forced covering or forced exits: arrow path labeled as forced and finite.
- No-trade: neutral watch token with incomplete-evidence glyph.

## Label Lanes

- Setup labels sit above the shared setup band.
- Outcome labels sit inside the top of each outcome panel.
- Price badges are monospace and aligned to the right edge of the panel.
- Decision-strip labels are short, no more than two lines.
- Rail rows use grouped evidence categories instead of full spec prose.

## Split Rule

Keep one plate when there are two outcomes and one no-trade decision strip.

Recommend split or separate composition approval if:

- no-trade requires the same visual weight as the two active outcomes;
- each outcome needs more than 2 callouts;
- shared setup cannot remain dominant;
- shared scale cannot be shown without shrinking branch evidence.

## Stop Conditions

Stop if:

- the burst or break is classified before the follow-on evidence;
- the two panels do not share setup or scale;
- covering flow is drawn as fresh demand;
- the defended branch is visually missing;
- no-trade is omitted when required by the spec;
- rails replace the body read;
- density forces type below the approved scale.
